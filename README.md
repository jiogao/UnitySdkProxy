# UnitySdkProxy

简化版聚合SDK工具。

实现Unity接入Android和iOS第三方SDK的中间层，方便已接入的sdk跨项目使用。
三个部分可以分别由不同开发人员开发，互不影响，实现sdk接入逻辑和游戏逻辑分离。
使用已接入sdk的固定打包工程引用Unity导出的纯净工程，Unity工程无需导入第三方库。
